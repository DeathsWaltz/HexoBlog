安装Docker-compose

使用github进行安装可能会下载失败，可先下载docker-compose安装文件后上传到服务进行安装。上传到服务器后移动到/usr/local/bin目录，使用chmod +x /usr/local/bin/docker-compose命令修改docker-compose为可执行权限。
